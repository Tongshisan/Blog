# css 宽度高度问题



**min-width     width     max-wdith    (height 同理)**

 思考: 下面的宽度是多少?

```css
div {
  min-width: 40px;
  width: 30px;
  max-width: 20px;
}
```

**答案: ** 40px



看实际的例子:

```html
<!--
 * @Author: tongshisan
 * @Date: 2020-10-13 16:36:44
 * @LastEditTime: 2020-10-13 16:44:27
 * @LastEditors: Please set LastEditors
 * @Description: In User Settings Edit
 * @FilePath: /Blog/CSS/宽度高度问题.html
-->
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
    <style>
        * {
            margin: 0;
            padding: 0;
        }
        html, body {
            display: flex;
            justify-content: center;
            align-items: center;
            width: 100%;
            height: 100%;
        }
        .test {
            min-width: 40px;
            width: 30px;
            max-width: 20px;
            height: 50px;
            background: #000;
        }
    </style>
</head>
<body>
    <div class="test"></div>
</body>
</html>
```



**运行结果显示 <div class="test"></div>  的大小为 40 * 50**
